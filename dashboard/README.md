# Assessment Dashboard

This is a vibe-coded HTML dashboard generated by Kiro using the Markdown report files in `../example-reports/`.

Each HTML page visualizes the assessment results for a specific goal:

| File | Goal |
|------|------|
| `index.html` | Landing page |
| `agentic-readiness.html` | Agentic Readiness (default equal-weight) |
| `agentic-usecase.html` | Enable Agentic Use Case |
| `cloud-native.html` | Cloud Native Modernization |
| `cost-optimization.html` | Cost Optimization |
| `cloudformation.yaml` | CloudFormation template for hosting on S3 + CloudFront |

To view locally, just open `index.html` in a browser — no build step needed.
