Detailed Findings

HIGH INF-Q5: No network security configuration defined

Category: Infrastructure & DevOps

Finding

No VPC, subnet, security group, NACL, or network segmentation configuration exists within this repository. Infrastructure deployment (including network topology) is handled entirely externally. The Docker Compose exposes all services on host ports without network isolation.

Recommendation

Define VPC architecture as IaC: private subnets for Druid data nodes, restricted subnets for coordination, and public-facing Router/Broker behind an ALB or API Gateway. Implement security groups with least-privilege rules between service tiers.

HIGH SEC-Q1: No audit logging infrastructure

Category: Security Baseline

Finding

No CloudTrail, audit trail, or immutable logging infrastructure is defined within this repository. Druid has application-level request logging but no audit-grade logging with immutability guarantees.

Recommendation

Define CloudTrail for API-level audit logging. Configure the request logger with log shipping to S3 with Object Lock enabled. Integrate CloudWatch Logs with retention policies.

HIGH SEC-Q2: No encryption at rest configured

Category: Security Baseline

Finding

No encryption-at-rest configuration exists. The default deep storage (local filesystem) and metadata store have no KMS encryption configured. S3 extension supports server-side encryption but no default enforces it.

Recommendation

Configure S3 deep storage with SSE-KMS using customer-managed keys. Define Aurora PostgreSQL with KMS encryption enabled. Add KMS key resources to IaC.

HIGH OPS-Q2: No SLO definitions

Category: Operations & Observability

Finding

No SLO definitions, error budgets, or formal service-level targets found. No CloudWatch alarm definitions for latency. Metrics framework exists but no SLOs are defined on top of them.

Recommendation

Define SLOs for: query p99 latency (<500ms), ingestion lag (<30s), metadata store availability (99.9%), and segment availability (99.99%). Implement as CloudWatch composite alarms with error budget dashboards.
