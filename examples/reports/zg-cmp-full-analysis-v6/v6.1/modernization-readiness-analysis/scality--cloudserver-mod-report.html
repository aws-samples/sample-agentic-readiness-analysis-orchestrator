Category: Infrastructure & DevOps
Finding

No VPC, subnet, security group, or network segmentation definitions exist in this repository. Health checks are IP-allowlisted but no infrastructure-level network isolation is defined.

Recommendation

Define VPC with private subnets, least-privilege security groups, and network segmentation in IaC. Consider VPC endpoints for AWS service access.

HIGH INF-Q10: No Infrastructure as Code

Category: Infrastructure & DevOps
Finding

No IaC exists in this repository. No Terraform, CloudFormation, CDK, Helm charts, or Kustomize manifests found. All infrastructure provisioning is managed externally.

Recommendation

Adopt CDK (TypeScript) or Terraform to define EKS cluster, DocumentDB/ElastiCache, VPC/networking, ALB, auto-scaling, and backup plans.

MEDIUM INF-Q2: Self-Managed Databases

Category: Infrastructure & DevOps
Finding

MongoDB replica set and Redis are self-managed. No managed database services (DocumentDB, ElastiCache) referenced. Operational burden includes manual patching, backup, and scaling.

Recommendation

Migrate MongoDB to DocumentDB or DynamoDB (preferred). Migrate Redis to ElastiCache or MemoryDB.

MEDIUM APP-Q2: Monolithic Application

Category: Application Architecture
Finding

Single deployable unit with identifiable modules (api, data, metadata, kms, auth) sharing a single process, config, and direct in-process function imports. Modules cannot be independently scaled or deployed.

Recommendation

Begin Strangler Fig decomposition. Extract metadata service first (already has wrapper interface).

MEDIUM SEC-Q5: No Dedicated Secrets Management

Category: Security Baseline
Finding

Secrets managed via Docker secrets, GitHub Secrets, and environment variables. No AWS Secrets Manager or rotation configured. Config files contain placeholder credentials.

Recommendation

Adopt AWS Secrets Manager for all production credentials. Configure automatic rotation. Use IAM roles where possible.

MEDIUM OPS-Q1: No Distributed Tracing

Category: Operations & Observability
Finding

No distributed tracing instrumented. No OpenTelemetry, X-Ray, or Jaeger. Debugging request flows across service boundaries requires manual log correlation.

Recommendation

Instrument with OpenTelemetry SDK for Node.js. Export traces to AWS X-Ray. Propagate W3C traceparent headers.
