Remediation: Deploy API Gateway with OAuth2 or API key authentication.

API-Q1: No documented API interface for agent tool binding High P0

HTTP API exists with 15+ endpoints but is internal-only with no formal documentation or versioning.

Remediation: Create OpenAPI specification documenting all routes and schemas.

All callers from allowed IPs have identical access to all API operations.

AUTH-Q3: No action-level authorization Medium P1

No RBAC/ABAC enforcement. A read-only agent could invoke POST endpoints.
