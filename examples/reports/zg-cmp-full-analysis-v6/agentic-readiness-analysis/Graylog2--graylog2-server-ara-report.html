All Findings (15)

HIGH AUTH-Q6: No Audit Logging in Open-Source Edition

The open-source edition ships with NullAuditEventSender — a no-op that discards all audit events. No audit trail exists for any operation.
Priority: P1Effort: MediumPhase: 1Category: Authentication & Authorization

MEDIUM AUTH-Q2: No Pre-Built Agent Role Template

Fine-grained permission model exists (100+ permissions) but no dedicated agent role template with minimal permissions pre-configured.
Priority: P1Effort: LowPhase: 1Safety Impact: Yes

MEDIUM AUTH-Q3: Admin Bypass Circumvents Action-Level Authorization

Admin role bypasses all action-level permission checks. Agent tokens created by admins could inherit unrestricted access.
Priority: P1Effort: MediumPhase: 1Safety Impact: Yes

MEDIUM AUTH-Q7: No Single-Operation Identity Suspension

Token revocation requires multi-step process. No single API call to immediately invalidate a specific agent token.
Priority: P1Effort: MediumPhase: 1Safety Impact: Yes

MEDIUM STATE-Q5: No API-Layer Rate Limiting

No rate limiting middleware, 429 responses, or rate limit headers. No defense against agent-induced traffic storms.
Priority: P1Effort: MediumPhase: 1Safety Impact: Yes

MEDIUM DATA-Q6: No PII Redaction in Logs

Plain-text logging with no PII scrubbing. User identifiers may appear in logs without redaction.
Priority: P1Effort: MediumPhase: 1Safety Impact: Yes

MEDIUM API-Q2: No Complete API Specification

Only one partial OpenAPI spec committed. Build-time generation exists but output not version-controlled.
Priority: P2Effort: LowPhase: 2

Strengths (Agent-Ready Aspects)

  • MCP Server — Native Model Context Protocol endpoint with 12 tools, permission-gated, configurable
  • Machine Identity — Access tokens with encryption at rest, configurable TTL, secure random generation
  • Fine-Grained Permissions — 100+ permissions with per-instance scoping via GRN capabilities
  • Structured JSON APIs — Well-defined DTOs, pagination support, consistent response format
  • Sensitive Data Exclusion — Password hashes excluded from API responses by architectural design
  • Event Webhooks — Outbound HTTP notifications with URL allowlisting for reactive agent patterns
  • Rich Metrics — 100+ Prometheus metrics with stable naming contract
