No CI/CD pipeline in the repository. All deployments are manual cdk deploy from a developer's machine.

OPS-Q5: Deployment StrategyHIGH
Category: Operations & Observability
Finding
No deployment strategy. Manual cdk deploy directly to production. No canary, blue/green, or traffic shifting.
Recommendation
Implement Lambda canary deployments with AutoPublishAlias and DeploymentPreference. Add CloudFormation rollback triggers.
OPS-Q6: Integration TestingHIGH
Category: Operations & Observability
Finding
Only test file is a placeholder with all assertions commented out. No integration, API, or contract tests.

Implement CDK assertion tests, API integration tests, and EventBridge flow tests using Jest.
INF-Q5: Network SecurityMEDIUM
Category: Infrastructure & DevOps
Finding
API Gateway endpoints publicly accessible without WAF protection or resource policies. Lambda uses IAM for service access (correct) but edge layer lacks protection.
Recommendation
Add AWS WAF to API Gateway stages. Configure resource policies. Add usage plans with throttling.
INF-Q8: Backup and RecoveryMEDIUM
Category: Infrastructure & DevOps
Finding
No backup configuration. DynamoDB tables use RemovalPolicy.DESTROY. No PITR. No AWS Backup plans.
Recommendation
Enable DynamoDB PITR on all tables. Change removalPolicy to RETAIN. Consider AWS Backup for scheduled backups.
APP-Q1: Programming LanguagesMEDIUM
Category: Application Architecture
Finding
Node.js 14.x (EOL), TypeScript 3.9.7, CDK 2.17.0 significantly outdated. AWS SDK v3 is current but runtime and framework are legacy.
Recommendation
Upgrade Lambda runtime to Node.js 20.x+. Upgrade CDK to latest 2.x. Upgrade TypeScript to 5.x.
SEC-Q3: API AuthenticationMEDIUM
Category: Security Baseline
Finding
No authentication on any API Gateway endpoint. All REST APIs open to the public internet without authorizers or API keys.
Recommendation
Add Cognito User Pool or Lambda authorizer to all API Gateway methods. Configure API keys with usage plans.
SEC-Q4: Centralized IdentityMEDIUM
Category: Security Baseline
Finding
No identity provider integration. No Cognito, OIDC, or SAML. No authentication system exists.
Recommendation
Integrate Amazon Cognito as centralized IdP with OAuth2/OIDC flows and API Gateway authorizers.
SEC-Q6: Compute HardeningMEDIUM
Category: Security Baseline
Finding
Lambda runtime Node.js 14.x is EOL. No vulnerability or dependency scanning configured.
Recommendation
Upgrade to Node.js 20.x+. Add Dependabot or Snyk for vulnerability scanning.
OPS-Q1: Distributed TracingMEDIUM
Category: Operations & Observability
Finding
No X-Ray active tracing on Lambda. No OpenTelemetry. No trace propagation across EventBridge/SQS/Lambda flow.
Recommendation
Enable X-Ray active tracing on all Lambda functions and API Gateway stages. Consider ADOT Lambda layer.
OPS-Q4: Anomaly DetectionMEDIUM
Category: Operations & Observability

No CloudWatch alarms configured. No anomaly detection. No incident notification integration.

Add alarms for Lambda error rate, duration p99, DynamoDB throttles, SQS DLQ count. Add SNS topic for notifications.
