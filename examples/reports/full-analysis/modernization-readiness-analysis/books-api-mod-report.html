MEDIUM SEC-Q7: Application Security Pipeline

Category: Security Baseline
Finding
No SAST, DAST, dependency scanning, or container scanning in CI/CD pipeline. Only unit tests run. No Dependabot, no Snyk, no SonarQube.
Recommendation
Add Dependabot or npm audit --audit-level=high to build phase. Integrate Semgrep or CodeGuru Reviewer for SAST.

MEDIUM INF-Q8: Backup and Recovery

Category: Infrastructure & DevOps
Finding
No backup or point-in-time recovery (PITR) configured for DynamoDB table. No AWS Backup plan defined.
Recommendation
Enable DynamoDB PITR: add PointInTimeRecoverySpecification: PointInTimeRecoveryEnabled: true to BooksTable.

MEDIUM APP-Q5: API Versioning Strategy

Category: Application Architecture
Finding
No API versioning. Endpoints at /books with no version prefix, headers, or annotations.

MEDIUM INF-Q7: Auto-Scaling

Category: Infrastructure & DevOps
Finding
DynamoDB SimpleTable uses default provisioned capacity (5 RCU/5 WCU) with no auto-scaling or on-demand billing.

Switch to on-demand billing (BillingMode: PAY_PER_REQUEST) or configure auto-scaling.
